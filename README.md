**Project Objectives**

Analyze sales, profit, and regional performance
Identify top-performing product categories
Study monthly and yearly sales trends
Generate business insights using data visualization

**Tools & Technologies Used**

Python
Pandas
Matplotlib
Power BI
Colab

**Key Features**

Data cleaning and preprocessing
Date transformation and trend analysis
Region-wise and category-wise sales analysis using pivot tables
KPI analysis for Sales and Profit
Interactive dashboard with filters and visual reports

**Key Insights**

The West region generated the highest overall sales
Technology category contributed the most revenue
Sales showed noticeable growth during Q4
The South region had comparatively lower performance

**Dashboard Components**

Sales KPI Cards
Monthly Sales Trend
Region vs Category Analysis
Top-Selling Products
Profit Distribution Charts
Interactive Filters (Region, Category, Year)

**Overall Report**

This project demonstrates practical skills in data cleaning, exploratory data analysis (EDA), business intelligence, and dashboard development. It highlights how data-driven insights can support strategic business decisions and improve performance analysis.

**Dataset**

Superstore Sales Dataset from Kaggle
